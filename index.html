公司简介

北京晨章数据科技有限公司成立于2022年，专注于数据库技术领域。核心团队成员来自微软亚洲研究院、VMware、小米、百度、阿里巴巴等知名公司。

我们每年开放研发实习岗位，已接收来自北京大学、清华大学、CMU、墨尔本大学等国内外高校的学生实习。

公司致力于打造具有独立知识产权的新一代企业级数据库解决方案，并获得多项荣誉与投资。

我们在2025年3月全面开源，欢迎关注：EloqKV，EloqSQL，EloqDoc

产品介绍

EloqKV - 分布式KV事务数据库

  • 支持分层存储持久化
  • 高性能，响应低至0.1ms
  • 水平扩展，兼容Redis API
  • 显著降低成本

EloqSQL - 分布式关系型数据库

  • 多写支持，解决MySQL写瓶颈
  • 低延迟分布式事务
  • 兼容MySQL，无需修改应用

EloqDoc - 分布式文档数据库

  • 支持多写和存算分离
  • 强事务支持，弹性扩展
  • 兼容MongoDB API
